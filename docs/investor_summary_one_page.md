# BLOCKID — Trust Layer for Web3 Wallets

**One-page summary for investors and partners**

---

## Problem

Crypto scams cause billions in losses every year. Users and exchanges cannot reliably detect scam wallets before interacting.

---

## Solution

BlockID provides a trust score for every wallet using:

- Graph clustering of transactions
- Scam propagation detection
- Explainable risk reasons
- Real-time warnings in Phantom wallet
- API for exchanges and marketplaces

**This prevents scams before they happen.**

---

## Technology

**Pipeline:**

> Transactions → Clustering → Propagation → ML → Trust Score → On-chain Oracle

BlockID publishes trust scores to Solana PDAs and exposes a FastAPI Trust API for partners.
